# Clinic Appointment Management System 🏥

##  Project Description

Desktop application developed in C# for managing patients, appointments, and clinic records.

The system provides a graphical interface that allows clinic staff to create, update, search, and manage information efficiently through a centralized database.

---

##  Technologies Used

- C#
- Windows Forms
- MySQL
- ADO.NET

---

##  Main Features

- Patient management
- Appointment scheduling
- Record creation and modification
- Database integration
- Search and filtering functionality
- User-friendly desktop interface

---

##  Key Concepts Applied

- CRUD operations
- Database management
- Desktop application development
- Event-driven programming
- Data validation
- User interface design
